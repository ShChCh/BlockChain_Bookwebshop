1.Open Ganache and set it as follow parameters: gas price 1
						gas limit 3000000000
						RPC Server http://127.0.0.1:8545

2.Open chrome browser and add plugin MetaMask, then import some account from Ganache into Metamask in the localhost:8545 private network

3.Open Terminal in Dapp directory and type following command. Then the browser will be opened automatically.
	truffle compile
	truffle migrate --reset
	npm run dev

4.Choose a valid account in MetaMask and refresh page.
